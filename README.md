 Assignment Objective
This project demonstrates the complete NLP pipeline including preprocessing, feature engineering, similarity calculation, and machine learning techniques.
⚙️ Technologies Used
- Python  
- NLTK  
- Scikit-learn  
 🔄 Steps Implemented

 ✅ 1. Text Preprocessing
- Converted text to lowercase  
- Removed punctuation  
- Tokenization  
- Stopword removal  
- Stemming  
- Lemmatization  
 ✅ 2. Feature Engineering
- Used TF-IDF (Term Frequency - Inverse Document Frequency)
 ✅ 3. Cosine Similarity
- Calculated similarity between 3 sentences  
- Helps to measure how similar two texts are  
 ✅ 4. Supervised Learning (Text Classification)
- Algorithm: Naive Bayes  
- Classified text into Positive and Negative  
 ✅ 5. Unsupervised Learning (Clustering)
- Algorithm: K-Means  
- Grouped similar sentences into clusters
-  📊 Output
- Preprocessed words  
- TF-IDF features  
- Cosine similarity matrix  
- Classification accuracy  
- Cluster labels
